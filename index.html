C++

History

C++ was invented by Bjarne
Stroustrup and his team at
Bell Labs in 1979.

It was meant to be an
improvement over C, another popular language.

C++ is used by a multitude of programmers worldwide.


Uses

The main benefit of C++ is its speed.

Applications such as YouTube, Windows, and even Google uses C++.

It's also the top language choice for many competitive programming contests, where speed is the essence.

Java

History

Java was created by
James Gosling at Sun
Microsystems.

It features a strong Object-Oriented Programming style, focused on objects and classes.

The main use of Java is for application programming.


Uses

An unique feature of Java is the Java Virtual Machine (JVM) Compiler.

While most languages translate code to machine langauge directly, Java first uses the JVM to translate Java code into something called Java bytecode.

Because of this, Java can be run on many systems without having to recompile the code.

This makes Java ideal for creating a wide variety of applications.
